{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Crane 是什么? Crane 是一个基于 FinOps 的云资源分析与成本优化平台。它的愿景是在保证客户应用运行质量的前提下实现极致的降本。 {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-chart-area" title="成本展示" url="./docs/" %}} Kubernetes 资源( Deployments, StatefulSets )的多维度聚合与展示。{{% /blocks/feature %}} {{% blocks/feature icon="fa-light fa-check" title="成本分析" url="./docs/" %}} 周期性的分析集群资源的状态并提供优化建议。 {{% /blocks/feature %}} {{% blocks/feature icon="fa-solid fa-screwdriver" title="成本优化" url="./docs/" %}} 通过丰富的优化工具更新配置达成降本的目标。 {{% /blocks/feature %}} {{< /blocks/section >}}
